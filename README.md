# 🚂 远行商人 (Merchant Game)

一个关于旅行与贸易的互动游戏，玩家扮演一名行商，在不同城市间穿梭，低买高卖，积累财富。

## 游戏特色

- 🏙️ **多城市贸易** - 在不同城市间旅行，每个城市有独特的商品和市场
- 📈 **动态经济系统** - 价格随供需波动，低买高卖赚取差价
- 🎒 **背包管理** - 有限的携带空间，合理规划货物
- ⚡ **随机事件** - 路途中可能遭遇各种事件
- 🏆 **成就系统** - 达成目标解锁成就

## 技术栈

- 纯前端 HTML5 + CSS3 + JavaScript (ES6+)
- 本地存储保存游戏进度
- 响应式设计，适配移动端

## 快速开始

1. 克隆仓库
2. 用浏览器打开 `index.html`
3. 开始你的商人之旅！
